## Install For Mac:
1.install homebrew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
2.install nodejs
```
brew update
brew install node
```
3.check nodejs version
```
node -v
```
4.git
```
brew install git
```

## Install For Windows:
